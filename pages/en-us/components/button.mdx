<Attributes edit="/pages/zh-cn/components/button.mdx">
<Attributes.Title>Button.Props</Attributes.Title>

| Attribute    | Description                   | Type                                            | Accepted values                   | Default   |
| ------------ | ----------------------------- | ----------------------------------------------- | --------------------------------- | --------- |
| **variant**  | button type                   | `ButtonVariants`                                | [ButtonVariants](#buttonVariants) | `default` |
| **color**    | button color                  | `ButtonColors`                                  | [ButtonColors](#buttontypes)      | `default` |
| **size**     | button size                   | `NormalSizes`                                   | [NormalSizes](#normalsizes)       | `medium`  |
| **loading**  | display loading indicator     | `boolean`                                       | -                                 | `false`   |
| **auto**     | autoscale width               | `boolean`                                       | -                                 | `false`   |
| **effect**   | display animation             | `boolean`                                       | -                                 | `false`   |
| **disabled** | disable button                | `boolean`                                       | -                                 | `false`   |
| **onClick**  | click handler                 | `MouseEventHandler`                             | -                                 | -         |
| **htmlType** | native type of button element | `ButtonHTMLAttributes.type`                     | -                                 | `button`  |
| **ref**      | forwardRef                    | <Code>Ref<HTMLButtonElement &#124; null></Code> | -                                 | -         |
| ...          | native props                  | `ButtonHTMLAttributes`                          | `'id', 'className', ...`          | -         |

<Attributes.Title>ButtonVariants</Attributes.Title>

```ts
type ButtonVariants = 'default' | 'solid' | 'text'
```

<Attributes.Title>ButtonColors</Attributes.Title>

```ts
type ButtonColors = 'default' | 'primary' | 'secondary' | 'success' | 'warning' | 'error'
```

<Attributes.Title>NormalSizes</Attributes.Title>

```ts
type NormalSizes = 'mini' | 'small' | 'medium' | 'large'
```

</Attributes>
