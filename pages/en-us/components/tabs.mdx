<Attributes.Title>Tabs.Props</Attributes.Title>

| Attribute        | Description                   | Type                    | Accepted values                 | Default |
| ---------------- | ----------------------------- | ----------------------- | ------------------------------- | ------- |
| **initialValue** | initial value                 | `string`                | -                               | -       |
| **value**        | controlled when value present | `string`                | -                               | -       |
| **onChange**     | change event                  | `(val: string) => void` | -                               | -       |
| **showDivider**  | if devider present            | `boolean`               | `true` or `false`               | `false` |
| **varient**      | Tab varient                   | `string`                | `solid` or `line`               | `line`  |
| ...              | native props                  | `HTMLAttributes`        | `'alt', 'id', 'className', ...` | -       |

<br />
<strong>Note:</strong>

1. `value` is key to tell if the component is controlled or not, do not switch between the two,
   we will not throw error like react though

<br />

<Attributes.Title>Tabs.useTabsHandle</Attributes.Title>

| Attribute         | Description                             | Accepted values | Default |
| ----------------- | --------------------------------------- | --------------- | ------- |
| **getCurrentTab** | act as a getter regarding to active tab | `void`          | -       |
| **setCurrentTab** | act as a setter regarding to active tab | `string`        | -       |

<br />
<strong>Note:</strong>

1. `ref` make sense only in **uncontrolled** component, i.e. `value` not present.
   `props` is the recommended way to sync UI in a typical react data flow.
   Choose `useTabsHandle` if you can really validate your usecase

<br />

<Attributes.Title alias="Tabs.Tab">Tabs.Item.Props</Attributes.Title>

| Attribute            | Description         | Type                          | Accepted values | Default |
| -------------------- | ------------------- | ----------------------------- | --------------- | ------- |
| **label**            | display tab's label | `string` or `React.ReactNode` | -               | -       |
| **value** (required) | unique ident value  | `string`                      | -               | -       |
| **disabled**         | disable current tab | `boolean`                     | -               | `false` |

</Attributes>
