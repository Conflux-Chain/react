## Pagination
A long list can be divided into several pages, and only one page will be loaded at a time.

<Attributes edit="/pages/en-us/components/pagination.mdx">
<Attributes.Title>Pagination.Props</Attributes.Title>

| Attribute                      | Description                                                                                               | Type                                     | Accepted values                           | Default                   |
|--------------------------------|-----------------------------------------------------------------------------------------------------------|------------------------------------------|-------------------------------------------|---------------------------|
| **total**                      | Total number of data items                                                                                | `number`                                 |                                           | 1                         |
| **pageSize**                   | Number of data items per page                                                                             | `number`                                 | -                                         | -                         |
| **page**                       | Current page number                                                                                       | `number`                                 | -                                         | 1                         |
| **defaultPage**                | Default initial page number                                                                               | `number`                                 | -                                         | 1                         |
| **variant**                    | Pagination variant                                                                                        | `string`                                 | [PaginationVariants](#paginationvariants) | `line`                    |
| **limit**                      | Limit of display page                                                                                     | `number`                                 | -                                         | 7                         |
| **size**                       | Pagination size                                                                                           | `NormalSizes`                            | [NormalSizes](#normalsizes)               | `medium`                  |
| **defaultPageSize**            | Default number of data items per page                                                                     | `number`                                 | -                                         | 10                        |
| **pageSizeOptions**            | Specify the sizeChanger options                                                                           | `string[]`                               | -                                         | ['10', '20', '50', '100'] |
| **labelPageSizeChangerBefore** | Customize the label before pageSizeChanger                                                                | `ReactNode` / `string`                   | -                                         | `SHOW`                    |
| **labelPageSizeChangerAfter**  | Customize the label after pageSizeChanger                                                                 | `ReactNode` / `string`                   | -                                         | `RECORDS`                 |
| **labelQuickJumperBefore**     | Customize the label before quickJumper                                                                    | `ReactNode` / `string`                   | -                                         | `GO TO`                   |
| **labelQuickJumperAfter**      | Customize the label after quickJumper                                                                     | `ReactNode` / `string`                   | -                                         | `PAGE`                    |
| **showQuickJumper**            | Determine whether you can jump to pages directly                                                          | `boolean`                                | -                                         | false                     |
| **showPageSizeChanger**        | Determine whether to show pageSize select                                                                 | `boolean`                                | -                                         | false                     |
| **onChange**                   | Called when the page number changes, and it takes the resulting page number and pageSize as its arguments | `(page: number,pageSize:number) => void` | -                                         | -                         |
| **onPageSizeChange**           | Called when pageSize is changed                                                                           | `(page: number,pageSize:number) => void` | -                                         | -                         |
| ...                            | Native props                                                                                              | `HTMLAttributes`                         | `'id', 'className', ...`                  | -                         |
