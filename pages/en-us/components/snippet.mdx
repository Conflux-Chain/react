## Snippet

Display a snippet of copyable code for the command line.

<Playground
  desc="Basic usage."
  scope={{ Snippet }}
  code={`
<Snippet text="yarn add @cfxjs/react-ui" width="300px" />
`}
/>

<Playground
  title="width"
  desc="Specify width manually."
  scope={{ Snippet }}
  code={`
<Snippet text="yarn add @cfxjs/react-ui" width="100%" />
`}
/>

<Playground
  title="types"
  desc="Show different states with colors."
  scope={{ Snippet, Spacer }}
  code={`
<>
  <Snippet text="yarn add @cfxjs/react-ui" color="dark" width="300px" />
  <Spacer />
  <Snippet text="yarn add @cfxjs/react-ui" color="success" width="300px" />
  <Spacer />
  <Snippet text="yarn add @cfxjs/react-ui" color="warning" width="300px" />
  <Spacer />
  <Snippet text="yarn add @cfxjs/react-ui" color="error" width="300px" />
  <Spacer />
  <Snippet text="yarn add @cfxjs/react-ui" color="secondary" width="300px" />
  <Spacer />
  <Snippet text="yarn add @cfxjs/react-ui" color="lite" width="300px" />
</>
`}
/>

<Playground
  title="Multi Line"
  scope={{ Snippet }}
  code={`
<Snippet text={['cd project', 'now']} width="300px" />

/>

<Playground

<Attributes edit="/pages/en-us/components/snippet.mdx">
<Attributes.Title>Snippet.Props</Attributes.Title>

| Attribute  | Description             | Type                            | Accepted values                  | Default   |
| ---------- | ----------------------- | ------------------------------- | -------------------------------- | --------- |
| **text**   | code snippet            | `string` `Array<string>`        | -                                | -         |
| **color**  | snippet colors          | [SnippetColors](#snippetcolors) | [SnippetColors](#snippetcolors)  | `default` |
| **filled** | filled style            | `boolean`                       | -                                | `false`   |
| **width**  | set CSS string          | `string`                        | -                                | `initial` |
| **copy**   | function of copy button | `CopyTypes`                     | `'default', 'slient', 'prevent'` | `default` |
| ...        | native props            | `HTMLAttributes`                | `'id', 'name', 'className', ...` | -         |

<Attributes.Title>SnippetColors</Attributes.Title>

```ts
type SnippetColors = 'default' | 'secondary' | 'success' | 'warning' | 'error' | 'dark' | 'lite'
```

</Attributes>
