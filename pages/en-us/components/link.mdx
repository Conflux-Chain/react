<Attributes edit="/pages/en-us/components/link.mdx">
<Attributes.Title>Link.Props</Attributes.Title>

| Attribute     | Description                    | Type                                            | Accepted values | Default |
| ------------- | ------------------------------ | ----------------------------------------------- | --------------- | ------- |
| **href**      | link url                       | `string`                                        | -               | -       |
| **plain**     | display colorless link         | `boolean`                                       | -               | `false` |
| **iconLeft**  | display icon on the left side  | `React.ReactNode`                               | -               | -       |
| **iconRight** | display icon on the right side | `React.ReactNode`                               | -               | -       |
| **underline** | display underline              | `boolean`                                       | -               | `false` |
| **block**     | display as a separate block    | `boolean`                                       | -               | `false` |
| **disabled**  | disable link                   | `boolean`                                       | -               | `false` |
| **ref**       | forwardRef                     | <Code>Ref<HTMLAnchorElement &#124; null></Code> | -               | -       |
| ...           | native props                   | `AnchorHTMLAttributes`                          | `'target', ...` | -       |

</Attributes>
