## Introduction

### About

`@conflux-chain/react-ui` is a React implementation for styles originating from <Link rel="nofollow" target="_blank" href="https://vercel.co/design">Vercel's design</Link>.

The design of **Vercel** is concise and aesthetic, this is an important reason for popularity of Vercel.
Now you can implement the styles in your own React project.

Conflux UI is an unofficial organization, all our projects are open source.
