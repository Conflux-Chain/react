<Attributes edit="/pages/zh-cn/components/loading.mdx">
<Attributes.Title>Loading.Props</Attributes.Title>

| 属性      | 描述         | 类型             | 推荐值                                                                                      | 默认      |
| --------- | ------------ | ---------------- | ------------------------------------------------------------------------------------------- | --------- |
| **color** | 指示器的颜色 | `string`         | `'default','primary','secondary', 'success','error','warning' '#121212', 'rgba(1,1,1,0.1)'` | `default` |
| **size**  | 指示器的大小 | `string`         | `'mini', 'small', 'medium', 'large','10px' ,'10rem'`,                                       | `medium`  |
| ...       | 原生属性     | `HTMLAttributes` | `'id', 'title', 'className', ...`                                                           | -         |

`color` 或 `size` 首先匹配预设值，否则按照 css 字符处理

color 预设值: `default`,`primary`,`secondary`,`success`,`error`,`warning`

size 预设值: `mini`, `small`, `medium`, `large`

</Attributes>
