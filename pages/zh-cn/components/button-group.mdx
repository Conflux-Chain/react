<Attributes.Title>ButtonGroup.Props</Attributes.Title>

| 属性         | 描述                   | 类型                   | 推荐值                            | 默认      |
| ------------ | ---------------------- | ---------------------- | --------------------------------- | --------- |
| **variant**  | 按钮类型               | `ButtonVariants`       | [ButtonVariants](#buttonVariants) | `default` |
| **color**    | 按钮颜色               | `ButtonColors`         | [ButtonColors](#buttontypes)      | `default` |
| **size**     | 按钮大小               | `NormalSizes`          | [NormalSizes](#normalsizes)       | `medium`  |
| **vertical** | 以垂直方式显示所有按钮 | `boolean`              | -                                 | `false`   |
| **disabled** | 是否禁用所有按钮       | `boolean`              | -                                 | `false`   |
| ...          | 原生属性               | `ButtonHTMLAttributes` | `'id', 'className', ...`          | -         |

<Attributes.Title>ButtonVariants</Attributes.Title>

```ts
type ButtonVariants = 'default' | 'solid'
```

<Attributes.Title>ButtonColors</Attributes.Title>

```ts
type ButtonColors = 'default' | 'primary' | 'secondary' | 'success' | 'warning' | 'error'
```

<Attributes.Title>NormalSizes</Attributes.Title>

```ts
type NormalSizes = 'mini' | 'small' | 'medium' | 'large'
```
