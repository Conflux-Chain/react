<Attributes.Title>Tag.Props</Attributes.Title>

| 属性         | 描述             | 类型                 | 推荐值                            | 默认      |
| ------------ | ---------------- | -------------------- | --------------------------------- | --------- |
| **variant**  | 按钮的类型       | `TagVariants`     | [TagVariants](#tagVariants) | `default` |
| **color**    | 按钮的颜色       | `TagColors`       | [TagColors](#tagtypes)      | `default` |
| **effect**   | 是否显示动画效果 | `boolean`            | -                                 | `false`   |
| **disabled** | 是否禁用按钮     | `boolean`            | -                                 | `false`   |
| **onClick**  | 点击事件         | `MouseEventHandler`  | -                                 | -         |
| ...          | 原生属性         | `SpanHTMLAttributes` | `'id', 'className', ...`          | -         |
