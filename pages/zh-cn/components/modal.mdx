<Playground
  desc="使用 `open` 属性控制何时显示对话框。"
  scope={{ Modal, Button, useState, Code, Text }}
  code={`
() => {
  const [state, setState] = useState(false)
  const handler = () => setState(true)
  const closeHandler = (event) => {
    setState(false)
    console.log('closed')
  }
  return (
    <div>
      <Button auto onClick={handler}>显示对话框</Button>
      <Modal open={state} onClose={closeHandler}>
        <Modal.Title>箭头函数</Modal.Title>
        <Modal.Subtitle>它不能用作构造函数</Modal.Subtitle>
        <Modal.Content>
          <Text p> <Code>yield</Code> 关键字通常不能在箭头函数中使用（除非是嵌套在允许使用的函数内）。因此，箭头函数不能用作函数生成器。</Text>
        </Modal.Content>
        <Modal.Action onClick={() => setState(false)}>放弃使用</Modal.Action>
        <Modal.Action variant="solid">明白了</Modal.Action>
      </Modal>
    </div>
  )
}
`}
/>

<Attributes.Title>Modal.Props</Attributes.Title>

| 属性                      | 描述                  | 类型               | 推荐值                                     | 默认      |
|--------------------------|-----------------------|-------------------|-------------------------------------------|----------|
| **open**                 | 打开或关闭对话框         | `boolean`         | -                                         | `false`  |
| **closable**             | 是否显示右上角的关闭按钮  | `boolean`         | -                                         | `false`  |
| **onOpen**               | 对话框打开的事件         | `() => void`      | -                                         | -        |
| **onClose**              | 对话框关闭的事件         | `() => void`      | -                                         | -        |
| **width**                | 对话框的宽度            | `string`          | -                                         | `30rem`  |
| **wrapClassName**        | 对话框的自定义样式类名    | `string`          | -                                         | -        |
| **disableBackdropClick** | 点击背景层时是否关闭对话框 | `boolean`         | -                                         | `false`  |
| ...                      | 原生属性                | `HTMLAttributes`  | `'autoFocus', 'name', 'className', ...`   | -        |
