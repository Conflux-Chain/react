<br/>

## Quick Start

1. run `yarn add @cfxjs/react-ui` or `npm i @cfxjs/react-ui` install it.

2. import into project:

```jsx
import { CfxProvider, CssBaseline } from '@cfxjs/react-ui'

const Application = () => (
  <CfxProvider>
    <CssBaseline /> // ---> normalize styles
    <AppComponent /> // ---> Your App Component
  </CfxProvider>
)
```
